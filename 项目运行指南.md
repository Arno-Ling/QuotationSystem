## 🐛 故障排查

### 问题1: 应用无法启动

**症状：** 运行uvicorn命令后报错

**解决方案：**
```powershell
# 检查端口是否被占用
netstat -ano | findstr :8000

# 如果被占用，结束进程
taskkill /PID <进程ID> /F

# 或使用其他端口
python -m uvicorn main:app --host 0.0.0.0 --port 8001 --reload
```

### 问题2: LLM API连接失败

**症状：** 测试返回 "Connection error"

**检查清单：**
1. 检查 `.env` 文件中的配置：
   ```env
   OPENAI_API_KEY=tp-cn0tro64vccki1wqosmy4d79v12alzqfnuv60ag1q5jvvwez
   OPENAI_MODEL=MiMo-V2.5-Pro
   OPENAI_BASE_URL=https://token-plan-cn.xiaomimomo.com/anthropic
   ```
2. 验证API key是否有效
3. 检查网络连接
4. 查看应用日志中的详细错误

### 问题3: 数据库连接失败

**症状：** "Access denied for user 'root'@'localhost'"

**解决方案：**
```powershell
# 测试数据库连接
python backend/test_db_connection.py

# 如果密码不对，参考：
# backend/MYSQL_PASSWORD_RESET_GUIDE.md
```

### 问题4: ChromaDB错误

**症状：** "An instance of Chroma already exists"

**解决方案：**
这是警告，不影响功能。如果需要清理：
```powershell
# 删除ChromaDB数据（谨慎操作）
Remove-Item -Recurse -Force backend/chroma_db
```
